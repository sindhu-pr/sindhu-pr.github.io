jemdoc is a python program which takes as input a text file, say abc.jemdoc and generates an html file as output, abc.html. The format of writing is much easier and thus one can concentrate more on the organization of the content. After typing in abc.jemdoc and saving it we just need to run $jemdoc abc in order to generate abc.html.
